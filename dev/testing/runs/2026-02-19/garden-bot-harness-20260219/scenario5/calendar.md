# Calendar

## February
- Clean and prep tools
- Source stepping stones, feature stone, and boulder seat for side yard
- Order plants: vine maple, oceanspray, red flowering currant, low Oregon grape

## March
- Prepare side yard beds — clear existing vegetation, amend soil with compost
- Install hardscape: set stepping stones, feature stone, boulder seat (do this before planting)
- Plant shrubs: oceanspray, red flowering currant, low Oregon grape
- Plant vine maple (bare root available through early spring)

## April
- Plant ferns: sword fern, deer fern (after last hard frost)
- Plant kinnikinnick groundcover
- Mulch all beds with leaf mulch or fine bark (2-3")
- Water new plantings weekly if dry

## May–September
- Water new plantings weekly through first summer (deep soak, not frequent sprinkle)
- No fertilizer needed for natives — avoid it

## October
- Reduce watering — fall rains should take over
- Enjoy vine maple fall color

## November–January
- Remove old sword fern fronds in late winter (before new fiddleheads)
- Native moss will colonize naturally in shaded areas — no action needed
