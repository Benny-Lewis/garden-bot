# Seasonal Calendar — Portland, OR (Zone 8b)

## January
- [ ] Order seeds for spring
- [ ] Plan garden layout for the year
- [ ] Prune dormant fruit trees and roses (late Jan)
- [ ] Check for storm damage after winter weather

## February
- [ ] Start seeds indoors: tomatoes, peppers, eggplant (mid-Feb)
- [ ] Prune summer-blooming shrubs
- [ ] Soil test if not done yet
- [ ] Apply dormant spray to fruit trees
- [ ] Clean and sharpen tools

## March
- [ ] Direct sow cool-season crops: peas, lettuce, spinach, radishes
- [ ] Start seeds indoors: squash, cucumbers (late March)
- [ ] Divide perennials
- [ ] Top-dress beds with compost
- [ ] Begin weeding — stay ahead of it early
- [ ] Last frost typically mid-March (watch forecasts)

## April
- [ ] Direct sow: carrots, beets, chard, kale
- [ ] Transplant cool-season starts
- [ ] Plant potatoes
- [ ] Fertilize established plants
- [ ] Mulch beds (2-3")
- [ ] Set up irrigation if needed

## May
- [ ] Transplant warm-season starts after last frost: tomatoes, peppers, squash
- [ ] Direct sow beans, corn, sunflowers
- [ ] Plant annual flowers
- [ ] Harden off seedlings before transplanting
- [ ] Watch for aphids and slugs (slugs are a Portland staple)

## June
- [ ] Harvest spring crops (lettuce, peas, radishes)
- [ ] Succession sow lettuce and beans
- [ ] Water deeply and consistently as dry season begins
- [ ] Stake tomatoes and tall plants
- [ ] Deadhead flowers to encourage reblooming

## July
- [ ] Deep water — Portland dry season is real (often no rain Jul–Sep)
- [ ] Harvest summer crops
- [ ] Start fall/winter crop seeds indoors: broccoli, cauliflower, kale
- [ ] Watch for powdery mildew in dry heat
- [ ] Mulch to retain moisture

## August
- [ ] Direct sow fall crops: lettuce, spinach, radishes, turnips
- [ ] Transplant fall brassica starts
- [ ] Continue deep watering
- [ ] Save seeds from open-pollinated varieties
- [ ] Order garlic for fall planting

## September
- [ ] Plant garlic and overwintering onions (late Sep–Oct)
- [ ] Sow cover crops in empty beds (crimson clover, fava beans)
- [ ] Begin reducing water as rains return
- [ ] Harvest and preserve summer crops
- [ ] Plant fall containers

## October
- [ ] Plant spring-blooming bulbs: tulips, daffodils, crocus, allium
- [ ] Finish garlic planting
- [ ] Clean up spent summer plants
- [ ] Rake and compost oak leaves (or leave some as mulch in Zone C)
- [ ] Winterize irrigation system
- [ ] Plant native trees and shrubs (fall is ideal in PNW)

## November
- [ ] First frost typically mid-November
- [ ] Protect tender plants with frost cloth
- [ ] Continue planting natives and bare-root plants
- [ ] Spread leaf mulch on beds
- [ ] Clean and store tools

## December
- [ ] Review the year — what worked, what didn't
- [ ] Browse seed catalogs
- [ ] Maintain compost
- [ ] Minimal garden work — enjoy the rest
- [ ] Plan any hardscaping projects for late winter
