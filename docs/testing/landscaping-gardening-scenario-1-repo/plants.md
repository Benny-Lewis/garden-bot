## How to Use

When you add a plant, fill in:
- **#**: Sequential number for easy reference
- **Variety**: Specific cultivar if known
- **Zone**: A (front yard), B (backyard sun), C (backyard shade)
- **Source**: Where you bought it (nursery, seed company, gift, etc.)
- **Status**: Planted, Sprouted, Established, Dormant, Struggling, Lost
